### Ajectives that end in -e match both genders
### 1. El Maestr*o* inteligent*e*
### 2. La masestr*a* inteligent*e*
### Many adjectives that end in a consonant match both genders
### 1. El amig*o* jove*n*
### 2. La amig*a* jove*n*
### Some adjectives that end in a consonant add -a to form the feminine singular. These exceptions have to be memorized 
### 1. El chic*o* trabajado*r*
### 2. La chic*a* trabajdo*ra*
